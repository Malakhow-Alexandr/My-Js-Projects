Processing the post form form.login-form must be event dispatched.
When submitting the form, the page should not reload. If there are blank fields in the form, warnings are displayed warning that all fields must be completed.
If the user has filled in all the fields and submitted the form, collect the field values in an object where the field name will be the property name and the field value will be the property value.
Output the object with the entered data to the console and clear the form field values using the reset method
